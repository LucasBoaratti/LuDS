# LuDS - Lucas Design System

A LuDS é uma biblioteca de componentes onde será testado diversos componentes como botões, inputs, checkboxes, radios e muito mais! Ela contém a biblioteca storybook na qual você pode testar seus componentes sem precisar rodar o app, apenas rodando em um lugar separado :D

Comandos importantes:

Instalar a pasta node_modules:

```bash
npm i
```

Instalar a biblioteca storybook:

```bash
npx storybook@latest init
```
Obs: clique em yes quando chegar no playwright :)

Rodar o storybook:

```bash
npm run storybook
```
